# NEXTEC. | Brutalist Creative Agency

An Awwwards-winning brutalist digital product studio template and ATS Dashboard, built with bleeding-edge web technologies. This project showcases premium, cinematic motion design, aggressive editorial typography, and robust frontend architecture.

## 🚀 Tech Stack
- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS v4
- **Animation:** GSAP & Framer Motion
- **Scroll Management:** `@studio-freight/lenis` (Smooth Scrolling)
- **Data Visualization:** Recharts
- **State Management:** Zustand
- **Icons:** Lucide React

## 🌟 Features
- **Cinematic Parallax & Staggers:** Advanced velocity-based scroll reveals via GSAP ScrollTrigger and Framer Motion.
- **Custom Global Cursor:** A performant GSAP `quickSetter` magnetic cursor with `mix-blend-mode` logic for immersive hover effects.
- **Brutalist UI System:** Oversized editorial typography, offset shadows, exposed borders, and a custom off-white/beige aesthetic (`#F3ECE7`).
- **Interactive Portfolio:** Horizontal scroll-hijacked gallery with shared layout animations (`layoutId`) for fullscreen modals.
- **ATS Dashboard System:** A complete, internal Linear-inspired Applicant Tracking System featuring Candidate search, status pipelines, Recharts analytics, and modular component architecture.

## 📦 Local Development

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

Built around the core principles of brutalism and digital editorialism:
- High contrast, raw geometric layouts.
- "Form follows function" but the form screams.
- Zero placeholder-looking components—every element feels bespoke.
